1. Namespace : release « {{ .Release.Name }} » dans « {{ .Release.Namespace }} ».
   Pour éviter le namespace default (bonnes pratiques / Kubescape) :
   helm upgrade --install {{ .Release.Name }} . --namespace task-manager --create-namespace

2. Vérifiez les pods : kubectl get pods -n {{ .Release.Namespace }} -l app.kubernetes.io/instance={{ .Release.Name }}

3. Ingress (si activé) : kubectl get ingress -n {{ .Release.Namespace }} {{ include "task-manager.fullname" . }}

4. Images attendues dans le cluster : {{ .Values.api.image.repository }}:{{ .Values.api.image.tag }} et {{ .Values.web.image.repository }}:{{ .Values.web.image.tag }}

5. Clé OpenRouter : --set openrouter.createSecret=true --set openrouter.apiKey=VOTRE_CLE
   ou openrouter.existingSecret=nom-d-un-secret-existant
